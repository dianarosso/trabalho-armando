Sistema de Gerenciamento de Mesadas PiggyBank

Resumo

Na atualidade percebe-se que de forma recorrente entre a população existe uma grande dificuldade em gerenciar a vida financeira. É comum observar que, as pessoas incentivadas pelo consumismo exacerbado, são levadas a gastar muito mais do que ganham. A mesada é uma ferramenta comum, usada pelos pais, para ensinar seus filhos sobre dinheiro e responsabilidade financeira. Por isso este trabalho tem como problema de pesquisa: a dificuldade de encontrar ferramentas que auxiliem na formação e gerenciamento de finanças pessoais. Diante disto e com a crescente importância da educação financeira na formação de indivíduos conscientes e responsáveis, torna-se cada vez mais relevante encontrar maneiras efetivas de ensinar habilidades financeiras desde a infância. O sistema gerenciador PiggyBank tem como proposta auxiliar na administração de mesadas, através de uma plataforma fácil de usar e interativa. Com ele, é possível controlar os valores a serem repassados, estabelecer metas de economia, definir limites de gastos, além de monitorar o uso do dinheiro pelos seus filhos ou dependentes.

Objetivo Geral e Objetivos Específicos

Neste sentido, este trabalho tem como objetivo, desenvolver um aplicativo de gerenciamento de mesadas para crianças para ajudar pais e filhos a gerenciar melhor suas finanças pessoais. Espera-se com os resultados deste trabalho, encontrar apontadores que avaliem a eficácia de um aplicativo de gerenciamento de mesadas para crianças, buscando entender seus benefícios e limitações e contribuir para o avanço da educação financeira desde a infância.

  • Sistema deve permitir cadastro de usuários;
  • Sistema deve permitir o login de usuários;
  • Sistema deve exibir o saldo do usuário;
  • Sistema deve ter acesso à internet;
  • Sistema deve permitir a alteração das informações pessoais;
  • Sistema deve permitir a exclusão de conta.
  • O sistema deve permitir a personalização do tema do aplicativo;
  • O sistema deve ser responsivo;
  • O sistema deve exibir o extrato do usuário;
  • O sistema deve permitir a criação de metas;
  • O sistema deve ter respostas rápidas para as ações do usuário.
